# Шортер ссылок

## Как работает

Есть фронтенд и бэкенд

На фронте: инпут с ссылкой, кнопка генерации
После внизу возвращается укороченная ссылка

На бэке:

POST /api/link

Body:

link - string

Ответ:

```json
{
  "link": "string"
}
```

БД не будет. За хранение отвечает Redis с TTL 4 часа

Если ссылка удаляется из Redis, возвращать 404
